# Unreal5_Study_Archive

This repository archives projects developed while taking an Unreal Engine 5 course on Udemy.

The projects are organized by course sections, and currently include three separate Unreal Engine projects.
The `repoSrc` folder is used to store additional study materials such as notes, screenshots, and troubleshooting records.

Udemy 강의(UE5) 수강 및 실습 과정에서 진행하는 **3개 프로젝트를 한 레포에 아카이빙**하는 저장소입니다.
각 프로젝트는 강의 섹션별로 분리해서 관리하고, 학습 과정에서 정리한 노트/스크린샷/트러블슈팅 기록은 별도 폴더에 누적합니다.
